# 키나의 성역 (AION 2 스케줄러) 프로젝트 가이드

## 기술 스택
- Frontend: React, Vite
- Database & Hosting: Firebase Firestore, Vercel
- 언어: JavaScript (JSX)

## 코딩 규칙 및 주의사항
1. 토큰 절약을 위해 응답 시 전체 코드를 다시 출력하지 말고, 수정된 함수나 로직 부분만 출력할 것.
2. 기능 추가 시 기존의 UI 디자인(CSS 스타일)과 데이터 구조를 최대한 망가뜨리지 않고 유지할 것.
3. Firebase 통신 시 항상 예외 처리(try-catch)를 꼼꼼히 작성할 것.
